## Запуск ковкерт

Для того чтобы запустить проект выполните команды:

```bash
npm i
```

```bash
npm run dev
```

#or

```bash
yarn
```

```bash
yarn dev
```

Откройте в браузере ссылку [http://localhost:3000](http://localhost:3000)

## Дальше
